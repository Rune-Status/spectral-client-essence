# ASM Library
This library provides a matcher ready extensions to the ASM 8.0.1 library.

## Installation
Add the Spectral maven repository
```groovy
repositories {
    maven { url "http://repo.spectralclient.org/" }
}
```
Add the dependency.
```groovy
dependencies {
    implementation "org.spectral:asm:0.0.1-r1"
}
```
